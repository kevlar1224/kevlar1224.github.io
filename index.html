Potential Symptoms of Poverty

The Data Collected

This project will provide a brief analysis of different population factors and see how strongly they are affected by poverty rates and each other. The factors considered are HIV/AIDS rates, infant mortality rates, insurance affordability program (IAP) application rates, and newborn disease rates.

As the datasets for each factor were collected from different years, this project should not be seen as an accurate analysis of how these factors correlate with one another, as populations change over time. Instead, this project can be taken as an example of data cleaning and potential analysis methods for multiple population statistics.

The various datasets were collected from data.world.
