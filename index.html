Latitude Analysis: Introduction

The purpose of this project is to analyze how much different factors of weather might be affected by latitude. To this end, we used a Jupyter Notebook and a weather api to check weather data from 500 randomly selected cities. The gathered data for cloudiness, humidity, temperature, and wind speed were all plotted against latitude.

It should be noted that there are fewer data points taken from the Southern Hemisphere, as the code was designed to ignore any coordinates that directed toward a point in the ocean. As most of the earth's land mass is in the Northern Hempishere, that is where most of the comes from.
